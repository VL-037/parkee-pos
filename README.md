## Features

- Parking Check In
- Parking Check Out (w/ voucher discount)

## How to run:

- Backend
  - run `mvn spring-boot:run` on `/backend`
  - APIs will be served on http://localhost:8080
  
- Frontend
  - run these commands
    ```
    npm install
    npm start
    ```
  - react-app can be access on http://localhost:3000
